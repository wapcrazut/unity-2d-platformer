# Unity 2D Platformer

2D Platformer prototype made with Unity 2022 LTS.

![Preview](./preview.gif)

## Features

- Player movement
- Player jump
- Enemy movement
- Collectable items
- Level exit
- Level restart
- Menu

## Controls

- `A` or `Left Arrow` - Move left
- `D` or `Right Arrow` - Move right
- `Space` - Jump
